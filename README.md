## 🔍 Understanding the Features
The application includes several tools to help you manage data.

### 🧪 Search Algorithms
The software uses three ways to find your data.
*   **HNSW:** This method organizes data in a graph to find items quickly. It provides high performance.
*   **KD-Tree:** This method uses a grid structure to group related information. It works well for small amounts of data.
*   **Brute Force:** This method looks at every item in your database. It is accurate but slower than other methods.

### 📈 Visualizing Data
The application displays your data as a chart. This feature maps your text into a two-dimensional space. You can watch as the software groups similar items together. This helps you understand how the database links different ideas.

### 🤖 Intelligent Chat
The app uses a local tool to process language. Paste any text into the box and the software adds it to your database. You can ask questions about your documents after you upload them. The app searches your database for the right answer.

## ⚙️ How It Works
The database translates text into lists of numbers. Computers process numbers better than words. When you ask a question, the software translates your question into these same numbers. It then compares your numbers against the numbers in your database. The closest matches show up first in your search results. This process makes the system fast and accurate.
